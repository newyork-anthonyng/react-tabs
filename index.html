react-tabs API

Tabs

A composite component that acts as the main container

onSelect(firstTabIndex, lastTabIndex)
This function is called whenever a tab is selected. The handler for this function will be passed the current tab index and last tab index.
<Tabs onSelect={(firstTab, lastTab) => console.log(`${firstTab}, ${lastTab}`)} />
selectedIndex
The tab with this index (zero-indexing) will be selected; defaults to 0.
<Tabs selectedIndex={1} />
forceRenderTabPanel
By default, react-tabs will only render the selected tab's contents. Setting this property to true allows you to override the default behavior, which may be useful in some circumstances (such as animating between tabs).
<Tabs forceRenderTabPanel={true} />
setUseDefaultStyles(false)
You can disable the default styling of react-tabs by calling this static method once.
Tabs.setUseDefaultStyles(false);
className
The class name to be added to the Tabs main component.
<Tabs className="foobar" />

TabList

A composite component that is the container for the Tab's.

className
The class name to be added to the TabList component.
<TabList className="foobar" />
activeTabClassName
A custom class name given to a Tab when it is active
<TabList activeTabClassName="myActiveClass" />
disabledTabClassName
A custom class name given to a Tab when it is disabled
<TabList disabledTabClassName="myDisabledClass" />

Tab

The actual tab component that the users will interact with. Selecting a tab can be done by either clicking with the mouse, or using the keyboard tab to give focus then navigating with the arrow keys (right/down to select the tab to the right of the selected, left/up to select the tab to the left of the selected. The children of the Tab will be shown as the label.

TabPanel

The content for the tab. There should be an equal number of Tab and TabPanel components. The Tab and TabPanel components are tied together by the order in which they appear. For example, when the first Tab is selected, the first TabPanel will be displayed.

className
The class name to be added to this component.
<TabPanel className="foobar" />
style
A style object to be applied to TabPanel
<TabPanel style={{border: "1px solid blue"}} />
